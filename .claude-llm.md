# Language Preferences

- **Idioma**: Español neutral (sin modismos)
- **NO usar**: "querés", "vos", "che", "boludo", etc.
- **USAR**: "quieres", "tú", "usted", formas neutrales de España/Latinoamérica

## Ejemplos

| Incorrecto (Argentina) | Correcto (Neutral) |
|------------------------|---------------------|
| ¿Querés cambiarlo? | ¿Quieres cambiarlo? |
| ¿Quéhicimos? | ¿Qué hicimos? |
| Está todo bien | Está todo bien |
| Dale, va | Okay,okay |

Esta preferencia se aplica a TODAS las respuestas en español.
